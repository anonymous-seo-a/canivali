## 2. 構成要素の意味

### 第1文: カードローンの定義 (主語=カードローン)
> カードローンは、個人を中心に法人も含めた幅広い資金調達の主要な選択肢である。

- **「個人を中心に法人も含めた」**: 範囲設定。個人事業主向け融資・役員貸付金等は範囲内。
- **「幅広い資金調達」**: V軸の広がり (カードローン以外の代替手段を含む) を許容。
- **「主要な選択肢」**: カードローンを中心軸として位置づけ、他V軸はその周辺に配置。

### 第2文: cardloanカテゴリの役割
> soicoのcardloanカテゴリは、検討者が自分の属性・条件・目的に合致する選択肢を、法令と一次情報および編集体制に裏付けられた信頼できる比較情報として提供する。

- **「属性・条件・目的」**: A軸 / B軸 / C軸の明示。subtopic 構造との整合。
- **「法令と一次情報」**: YMYL の暗黙シグナル。違法業者排除と一次情報重視を含意。
- **「編集体制に裏付けられた」**: E-E-A-T のうち Authoritativeness / Trustworthiness の宣言。
- **「比較情報」**: D軸 (ピラー) の中心的な機能。

### 第3文: 3層応答 + 違法警告 + 送客
> 検索意図の顕在・潜在・安心の3層すべてに閉合的に応答し、違法業者や悪質サービスは明確に警告して排除し、登録貸金業者への適切な送客を実現する。

- **「3層すべてに閉合的に応答」**: 真=美の閉合性テストの SEO 領域での具体化。CVR 5-10% の構造的根拠。
- **「違法業者や悪質サービスは明確に警告して排除」**: V16 (違法警告) の存在意義を定義。警告対象として記事化するが、送客対象から除外。
- **「登録貸金業者への適切な送客」**: 事業価値の明示。読者保護とのバランス。

## 3. 適用範囲

### 3.1 範囲内 (cardloan/ で扱う)
- 個人向け資金調達手段全般 (カードローン / キャッシング / 消費者金融 / 銀行ローン / 質屋 / 後払い等)
- 個人事業主・自営業向けの融資 (ビジネスローン / ファクタリング / 公庫融資)
- 役員貸付金等の法人会計でも資金調達文脈の話題
- 公的支援制度 (生活保護・生活福祉資金等)
- 違法警告 (闇金・先払い買取等) ※ただし送客対象から除外

### 3.2 範囲外 (cardloan/ から隔離)
- 仮想通貨・DeFi 等の投資手段
- 純粋な住宅ローン (借入だが資金調達ではなく住宅取得手段)
- 賃貸契約関連
- 純粋なクレカショッピング機能 (キャッシング枠は範囲内)

## 4. カテゴリ汚染判定基準

各記事を「範囲内 / 範囲外」に分類するための具体的判定木:

```
1. 内容が「資金が必要な個人または法人が、その資金を調達する手段」を扱っているか?
   YES → 2へ
   NO → 範囲外

2. 主たる選択肢が以下のいずれかか?
   - 借入 (担保有・無問わず)
   - 公的支援
   - 個人間借入
   - 売掛金売却 (ファクタリング)
   - 商品売却を伴う即時現金化
   YES → 範囲内
   NO → 範囲外

3. 反商品 (違法・悪質) であっても警告コンテンツとして範囲内
   ※ただし送客対象から除外
```
